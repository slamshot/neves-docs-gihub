# 流程设计器

​		基于flowable流程引擎封装的流程快速开发设计

## 开发

​		流程开发的入口

### 模型管理

​		流程模型开发管理、支持单个添加、批量导入、导出、流程发布、取消发布、删除、查看xml、查看流程图功能

​		![模型管理](..\images\flow\模型管理.png)

#### 流程设计

​		流程开发主界面

##### 组件

​		展示所有流程节点类型，以拖拉拽的形式放入设计区
